# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 개요

「Do it! 자료구조와 함께 배우는 알고리즘 입문 - 파이썬 편」 교재를 따라 공부하는 저장소. Python Jupyter 노트북으로 작성되며, 주석과 설명은 한국어로 작성한다.

## 구조

각 챕터는 `ChapXX.ipynb` 형식의 노트북 파일 하나로 구성된다 (예: `Chap01.ipynb`). 노트북 안에 알고리즘 구현과 테스트 케이스가 포함되어 있다.

## 실행 방법

Jupyter 또는 호환 환경(VS Code, JupyterLab)에서 실행. 외부 라이브러리 없이 Python 표준 라이브러리만 사용한다.

## 컨벤션

- 주석과 출력 문자열은 한국어로 작성한다.
- 각 알고리즘은 독립적인 함수로 구현하고, 다양한 입력 케이스로 테스트한다.
- 노트북 내에서 챕터 내용을 순서대로 진행한다.

## 대화 규칙

- 모든 대화는 한국어로 진행한다.
- 사용자가 교재 내용에 대해 질문하면 자세하고 친절하게 설명한다.
